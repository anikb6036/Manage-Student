# 🎓 Coaching Center Management Platform

A modern Coaching Center Management Platform that helps coaching institutes manage students, teachers, courses, attendance, fees, and reports efficiently.

## 🚀 Features

### 👨‍🎓 Student Management
- Add, edit, and delete students
- Student profile management
- Course enrollment
- Performance tracking

### 👨‍🏫 Teacher Management
- Manage teacher profiles
- Assign courses and batches
- View schedules
- Attendance tracking

### 📚 Course & Batch Management
- Create and manage courses
- Batch allocation
- Subject management
- Timetable scheduling

### 📝 Attendance Management
- Student attendance tracking
- Teacher attendance records
- Attendance reports

### 💰 Fee Management
- Fee collection
- Payment history
- Due fee tracking
- Receipt generation

### 📊 Dashboard & Reports
- Student statistics
- Revenue reports
- Attendance analytics
- Performance reports

### 🔐 Authentication & Authorization
- Secure login system
- Role-based access control
- Admin, Teacher, and Student roles

## ⚙️ Installation

### Clone the Repository

```bash
git clone https://github.com/your-username/coaching-center-management.git
```

### Navigate to Project Folder

```bash
cd coaching-center-management
```

### Install Dependencies

```bash
npm install
```

### Configure Environment Variables

Create a `.env` file in the root directory:

```env
PORT=5000
MONGODB_URI=your_mongodb_connection_string
JWT_SECRET=your_secret_key
```

### Run the Application

```bash
npm run dev
```

The application will run on:

```bash
http://localhost:5000
```
